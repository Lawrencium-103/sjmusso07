Project Scope

End-to-end design, development, and deployment of a private alumni portal featuring member management, role-based access, a full election/voting system with countdown phases, attendance & payments tracking, news publishing, and an AI-powered Yoruba chatbot assistant with voice I/O.

Deliverables

Frontend

  • Next.js 15 App Router — 29 routes
  • Responsive mobile-first UI (Tailwind CSS)
  • Premium design system: glass-morphism, staggered animations, brand palette (#054ea4 / #e7b801 / #d52419)
  • Public homepage, auth-gated dashboard & admin panel
  • 3-step voting flow with CSS confetti celebration
  • ElectionCountdown with auto 3-phase timer
  • Adunni chatbot — female Yoruba voice, Web Speech API, 18-intent knowledge base, GDPR/NDPR-compliant
  • Mobile horizontal nav on admin & dashboard

Backend & Infrastructure

  • SQLite → Neon (PostgreSQL) migration
  • 15 async API routes with $1-placeholder queries
  • Auth system with sessions & role-based guards
  • 42 alumni, 7 positions, 21 aspirants, 5 meetings, 9 news, 210 attendance & 252 payment records seeded
  • Public results API with scope-based visibility
  • Vercel serverless deployment (sjmusso07.vercel.app)
  • GitHub version control
